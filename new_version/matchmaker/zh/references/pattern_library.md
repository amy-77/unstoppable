# 冲突模式库

## 入场时机错误（新增）

**模式定义**：求偶者当前状态不适合进入市场（心理未修复/核心资产未积累/结构性劣势过重），强行入场本身就是核心冲突。系统应识别出"暂停入场、专注增值"是最优策略，而非默认给出市场匹配方案。

**触发条件**：

- 结构性劣势叠加过重（多重硬伤同时存在），入场ROI极低
- 心理创伤未愈（前段关系重创未修复），入场必然重复旧模式
- 核心资产尚未积累到入场门槛（收入/学历/身材等关键项存在可短期改善的硬伤）

**策略方向**：暂停入场 + 定向增值（收牌而非打牌）

**典型表现**：

- **AY104**（结构性劣势叠加）：多重硬伤同时存在，应先集中资源定向增值，待核心短板补足后再入场
  -> 暂停入场，制定6-12个月增值计划，优先解决ROI最高的短板项。
- 心理创伤未愈案例：前段关系造成深度心理损伤，当前状态入场只会吸引同类型对象或重复旧模式
  -> 先完成心理修复（专业咨询/自我疗愈），建立健康关系认知后再入场。

## 恋人属性缺失型困境（新增）

**模式定义**：区别于"情绪价值低"（偏男性、偏输出能力不足），本模式聚焦于"互动吸引力缺失"——求偶者硬件优秀但不会制造情感张力、不会撒娇、不会主动推进关系、缺乏恋人属性。核心问题不在渠道或标准，而在情感互动能力本身。

**触发条件**：

- 硬件条件优秀（外形/学历/家境均在线）但长期无法进入亲密关系
- 社交能力正常但缺乏异性吸引力/情感张力
- 被评价为"好人""优秀"但不被当作恋爱对象

**策略方向**：情感互动能力训练（而非渠道调整/标准管理）

**典型表现**：

- **AY165**（温室高配被动女）：全维度优秀但不会撒娇、不会制造暧昧、不会主动推进，导致异性只将其视为"好女孩"而非"想追的女孩"
  -> 核心训练：学会释放女性魅力信号、制造情感张力、主动创造亲密互动机会。问题不在市场端，在自身恋人属性的激活。

## 低分案例锚定摘要（case_digest补充）

以下案例为历史评分最低案例，记录其正确归因方向，防止同类错误重复：

- **AY104**：核心冲突应归因为"结构性劣势叠加导致不应入场"，正确策略节奏为"收牌增值"而非"打牌匹配"。系统常见误判：默认给出渠道/匹配方案，忽视暂停入场的必要性。
- **AY144**：核心冲突应归因为"价值不对称情境下的博弈姿态错误"，当求偶者已锁定远超自身条件的对象时，策略应从"平等筛选"切换为"锁定资源、降低姿态、全力捆绑"。系统常见误判：仍按平等博弈逻辑给出筛选建议。
- **AY165**：核心冲突应归因为"恋人属性/情感互动能力缺失"（内在人格短板），而非市场结构问题。正确策略为情感能力训练（学会撒娇、制造张力），而非渠道调整或标准放宽。系统常见误判：归因为外部市场问题。
- **AY178**：核心冲突应归因为"情感细腻度缺失"而非表面标签（如离异）。高物质掌控力男性的真正短板是情感供给能力不足，策略应聚焦内在人格改造。系统常见误判：将问题归因为离异标签的市场歧视。

**归因优先级原则**：当案例同时存在外部市场结构问题和内在人格短板时，优先归因内在人格短板（可改变、ROI更高），再处理外部结构问题。

**策略节奏判断原则**：
- 打牌（进攻型/市场适配型）：适用于核心资产已就位、心理状态健康、市场定位清晰的求偶者
- 收牌（防御型/自我改造型/暂停型）：适用于结构性劣势过重、心理未修复、核心能力未积累的求偶者
